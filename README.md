# Visualizador de Recursão com Flask

Projeto desenvolvido em Python utilizando Flask para demonstrar visualmente o funcionamento de algoritmos recursivos.

## Funcionalidades

O sistema possui três demonstrações de recursão:

### Torres de Hanói

* Escolha a quantidade de discos.
* Visualize todos os movimentos necessários para resolver o problema.
* Entenda o processo recursivo passo a passo.

### Inversão de String

* Digite um texto.
* Veja como a recursão inverte cada caractere.
* Acompanhe o histórico completo das chamadas e retornos da função.

### Conversão Decimal para Binário

* Informe um número decimal.
* Visualize as divisões sucessivas por 2.
* Entenda como o número binário é montado durante o retorno da recursão.

## Como Executar

### 1. Clone o repositório

```bash
git clone URL_DO_REPOSITORIO
```

### 2. Acesse a pasta do projeto

```bash
cd nome-do-projeto
```

### 3. Instale o Flask

```bash
pip install flask
```

### 4. Execute a aplicação

```bash
python app.py
```

### 5. Abra no navegador

```text
http://127.0.0.1:5000
```
